Development
I decided to create a web application that is dedicated to lipsticks. There are many e-comerce platforms that sell lipsticks however they lack usability principles. In addition there is no filtering or sorting options dedicated to this product. My goal with this web applciation is to create a space for an easy shopping experience and user friendly navigation. With this web application it is easy to click an item and add the item to the card while seeing the shopping card at all times.
